TRANSFOG ANNOTATION SERVER
==========================

  What is it?
  -----------

  TRANSFOG Annotation Server is an ontology-aware DAS annotation server based on GFF files.
  For examples of usage please see the src/web directory.

  Requirements
  ------------

  To install:
   o  A Java 1.4 or later compatible virtual machine for your operating system.
   o  Apache Ant 1.5.1 or later

  To run:
   o  A Java 1.4 or later compatible virtual machine for your operating system.
   o  A Servlet API 2.2 compatible Servlet Engine or J2EE Application Server.
